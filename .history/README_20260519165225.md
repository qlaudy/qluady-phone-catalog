# Qluady Phone Catalog

## Project Description
Qluady Phone Catalog is a responsive web application that displays a catalog of phones with filtering, sorting, and detailed product pages. The project is built as a learning assignment to practice React, TypeScript, routing, and working with API-like data structures.
