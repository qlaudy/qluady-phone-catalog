# Qlaudy Phone Catalog

## Overview
Welcome to the **Qlaudy Phone Catalog**, a modern e-commerce web application built with React and TypeScript.
The project is designed according to a Figma layout and focuses on building a clean, responsive and interactive user experience.

It includes a product catalog, product details pages, sorting, filtering, favorites, and shopping cart functionality.

---

## Demo
You can view a live demo of the application here:
https://qlaudy.github.io/qluady-phone-catalog/

---

## Key Features

### Product Catalog
- Displays a list of phones with detailed information
- Pagination for easy navigation between pages
- Sorting by price, newest, and alphabetically

### Product Details Page
- Full product information
- Image gallery with switching between photos
- Recommended products section

### Shopping Cart
- Add / remove products from cart
- Quantity management
- Total price calculation

### Favorites
- Add/remove products to favorites
- Persistent state (saved between sessions)

### Search & Filtering
- Search products by name
- Filter by category and other parameters

### UI / UX Features
- Responsive design for all devices
- Smooth transitions and hover effects
- Clean and modern UI based on Figma design

## Challenges

During development, several important challenges were solved:

### State Management
Managing complex state for cart, favorites, and product data across multiple pages.

### Routing
Implementing React Router for navigation between catalog, product details, cart, and favorites pages.

### Data Handling
Handling product data efficiently, including filtering, sorting, and pagination logic.

### UI Consistency
Ensuring consistent design across all components based on the Figma layout.

### Responsive Design
Adapting layout for mobile, tablet, and desktop screens while maintaining usability.

## Installation & Setup

To run this project locally, follow these steps:

### Clone the repository:
```bash
git clone https://github.com/qlaudy/qluady-phone-catalog.git
cd qluady-phone-catalog
```

### Install dependencies:
```bash
npm install
```

### Run the project locally:
```bash
npm start
```
